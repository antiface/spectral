spectral.core.source.usrp_n210 module¶

class spectral.core.source.usrp_n210.UsrpN210(addr, samp_freq=25000000.0, center_freq=2400000000.0, lo_offset=12500000.0, gain=10, sample_format='fc32')[source]¶

Bases: spectral.core.source.source.Source

Signal coming from a USRP N210 radio. This class starts a connection with the radio and provides methods for receiving samples, setting center frequency and gain values. The local oscillator offset is needed because the automatic dc offset calibration does not work with the finite_acquisition function.

Args:
addr: IP address of the USRP samp_freq: Sample frequency of the USRP center_freq: Initial center frequency on initialisation lo_ffset: Local oscillator offset gain: Amount of antenna gain (dB) sample_format: Output datatype of the USRP
